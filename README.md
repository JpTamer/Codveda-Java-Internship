# Java Development Internship - Codveda Technologies

This repository contains my technical submissions for the Java Development Internship at Codveda Technologies (April 2026 - May 2026). The project is structured into three levels, progressing from core logic to advanced system architecture.

## Project Overview
During this internship, I am focusing on building robust, scalable, and well-documented Java applications. Each task is implemented with a focus on **SOLID principles**, **clean code**, and **exception handling**.

## Project Structure

### Level 1: Basic Logic
* **Task 1: Basic Calculator** - A console-based arithmetic engine with comprehensive error handling for mathematical and input exceptions.
* **Task 2: Number Guessing Game** - Interactive CLI game utilizing the `Random` class and input validation logic.

### Level 2: OOP & Data Persistence
* **Task 1: Employee Management System** - A CRUD application utilizing Object-Oriented principles and the Java Collections Framework.
* **Task 2: Simple Banking Application** - Simulation of banking transactions focusing on encapsulation and custom exception handling.

### Level 3: Advanced Systems
* **Task 1: Library Management (JDBC)** - A database-driven system using JDBC for persistent storage and transaction management.
* **Task 2: Multithreaded Chat App** - A real-time networking application using Java Sockets and Concurrency (Threads) for multi-client communication.

## 🛠️ Tech Stack
* **Language:** Java (JDK 17+)
* **Database:** MySQL / PostgreSQL (for JDBC tasks)
* **Tools:** Git, GitHub, IntelliJ IDEA
* **Concepts:** OOPs, Exception Handling, Collections, JDBC, Multithreading, Socket Programming
